# MFcookieodd - 微信密友隐藏插件

一款专注于隐私保护的微信插件，可以将指定好友/群聊从微信各个位置完全隐藏。

## 功能介绍

### 核心功能
- **会话隐藏**：从微信首页、转发页隐藏指定好友/群聊的会话
- **通讯录隐藏**：从联系人列表中隐藏指定好友
- **朋友圈隐藏**：隐藏密友的朋友圈动态、互动消息
- **消息屏蔽**：屏蔽密友的所有消息提示（文字、语音、视频通话）
- **搜索过滤**：从搜索结果中过滤密友相关内容
- **收藏过滤**：从收藏列表中过滤密友相关内容

### 朋友圈隐私
- **部分可见添加密友**：发朋友圈选择"部分可见"时自动添加密友
- **不给谁看添加密友**：发朋友圈选择"不给谁看"时自动添加密友
- **公开默认对密友不可见**：发公开朋友圈时自动对密友不可见

### 临时查看
- **长按触发**：长按底栏「微信」临时显示密友（可自定义时长）
- **连点触发**：连续点击底栏「微信」临时显示密友（可自定义次数）
- **Face ID 验证**：可选开启 Face ID 验证后才能临时查看
- **自动恢复**：切换到后台或锁屏后自动恢复隐藏状态

### 其他功能
- **清空互动消息**：长按/点击底栏「发现」清空朋友圈互动消息和红点
- **设置入口保护**：可选开启 Face ID 保护插件设置入口

## 使用说明

### 进入插件设置
1. 打开微信 → 我 → 设置 → 关于微信
2. 连续点击版本号 5 次
3. 如开启了设置入口 Face ID，需验证后进入

### 添加/移除密友
1. 点击「添加密友」或「添加密群」按钮
2. 选择要隐藏的好友/群聊（支持多选）
3. 点击「完成」保存
4. 左滑密友/密群条目可移除

### 临时查看密友
- **长按模式**：长按底栏「微信」图标（默认 1 秒）
- **连点模式**：连续点击底栏「微信」图标（默认 3 次）
- 切换到后台或锁屏后自动恢复隐藏

### 清空朋友圈互动消息
- 长按/点击底栏「发现」图标即可清空

### 触发方式配置
1. 在插件设置中找到「临时查看触发方式」
2. 选择「长按触发」或「连点触发」
3. 打开开关时会弹窗设置具体参数

## 安装方式

### 越狱设备
将 `.deb` 文件通过 Filza 或 Cydia 安装

### 免越狱设备
使用全能签、轻松签等工具将 `MFcookieodd.dylib` 注入微信 IPA

## 兼容性

- iOS 13.0+
- 微信 8.x
